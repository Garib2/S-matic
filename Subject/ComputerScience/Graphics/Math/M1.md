## 1. INTRO.
### 1.1 MOTIVATION
- computing with points, vectors, and matrices
- Coordinate-based methods: 3가지 관점으로 해석될 문제
- transforming a 2D point(C)
```C
float P[2];
float PPrime[2];
float M[2][2];
P[0] = x;
P[1] = y;
M[0][0] = 3; M[0][1] = 0;
M[1][0] = 0; M[1][1] = 2;
PPrime[0] = P[0] * M[0][0] + P[1] * M[1][0];
PPrime[1] = P[0] * M[0][1] + P[1] * M[1][1];
```
